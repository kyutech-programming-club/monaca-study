## Description

## Features
- ユーザー（新規登録・ログイン・ログアウト）
- 言語（例：　C, Python）登録
- 要素（例：　printf, scanf）登録

## Requirement
- [monaca](https://ja.monaca.io/)
- [ニフクラ mobile backend](https://mbaas.nifcloud.com/)

## Usage
1. 新規登録orログイン
2. 言語選択or登録
3. 要素選択or登録
4. 「みんな」から、データベース上の言語・要素を確認
